# DragView

Drag view library for Android

# Screenshot

![DragView Screenshot](Screenshot.gif)

# Setup

* In your `build.gradle` :

```gradle
repositories {
    maven { url "https://jitpack.io" }
}

dependencies {
    compile 'com.github.nanjingboy:DragView:1.0.0'
}
```
